#### Configuration

```
# Install Required Tools.

chmod 755 ./Utils/InstallTools.sh
sh ./Utils/InstallTools.sh
```

```
# Install Python Libraries.

chmod 755 ./Utils/InstallPackages.sh
sh ./Utils/InstallPackages.sh
```
